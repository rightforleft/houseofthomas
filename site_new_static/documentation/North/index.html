Super Slides

Super Slides, is a full screen slider. you can see it on full screen demos.

HTML;

Javascript:
Standart Using:
$('#fullscreen , .p-section').superslides({
	play: 12000,
	animation: 'fade',
	inherit_height_from: window,
});
Auto Play False and Slide animation:
$('#fullscreen , .p-section').superslides({
	play: false,
	animation: 'slide',
	inherit_height_from: window,
});

Box Sliders

We've included Flex Slider and Owl Carousel Slider
Owl Carousel Slider;
 $(".home-boxes").owlCarousel({
	items : 3,
	// Responsive Settings
	itemsDesktop : [1169,3],
	itemsDesktopSmall : [979,2],
	itemsTablet : [600,1],
	itemsTabletSmall : false,
	itemsMobile : [560,1],
	// End Responsive Settings
	slideSpeed : 400
});

You can edit to it on js/plugins.js file for visible item numbers, and slide speeds.

 });

Contact Forms

You can use your domain's settings or Gmail settings.

Edition;
$mail->Host     = "smtp.domain.com"; 				 // SMTP server
$mail->SMTPAuth = true;    					 // turn on SMTP authentication
$mail->Username = "no-reply@domain.com"; 		         // SMTP username
$mail->Password = "secret";					 // Password

$mail->From     = "no-reply@domain.com";			 // SMTP username again
$mail->AddAddress("your-email@domain.com");		  	 // Your Adress
$mail->Subject  =  "New mail from NORTH !";
Gmail;
$mail->Host     = "smtp.gmail.com"; 							 // SMTP server for Gmail
$mail->SMTPAuth = true;    	
$mail->SMTPSecure = 'tls';								
$mail->Username = "your-gmail-address@gmail.com"; 				 // Your Mailer Gmail Address
$mail->Password = "secret";								 		 // Password
$mail->Port = 587;	
$mail->From     = "your-gmail-address@gmail.com";				 // Your Mailer Gmail Address (Same With Username)
$mail->AddAddress("your-address@domain.com");			  		 // Your Adress
$mail->Subject  =  "New mail from North !";
